# Incident Workflow

Use Issues to review anomalies and decide what needs action.

## Statuses

- `active`
- `acknowledged`
- `snoozed`
- `suppressed`
- `resolved`

## What You Can Do

- review the highest-priority issue first
- acknowledge work in progress
- snooze or suppress known noise
- resolve with a reason
- add notes and use bulk actions

## Recommended Flow

1. Open Issues.
2. Sort by priority and severity.
3. Check the baseline and explanation.
4. Add a note if needed.
5. Acknowledge, snooze, suppress, or resolve.
